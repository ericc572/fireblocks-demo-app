## Overview
This demo showcases the functionality of connecting a wallet via Metamask, then displaying the:
- Wallet Address
- Account Balance
- Latest transaction hash if at all. Displays none if does not exist.

## Technologies utilized
- wagmi.sh
- Next.js
- ethers.js
- Etherscan API
